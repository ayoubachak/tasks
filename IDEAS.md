# Product Ideas & Future Enhancements

This list captures feature concepts and improvements for future iterations. Items are intentionally high-level so they can be groomed into actionable tasks later.

## 1. Collaboration & Multi-User
- Real-time presence indicators in tasks and notes.
- Shared workspaces with granular permissions (read/comment/edit).
- Comment threads with @mentions and email/slack notifications.

## 2. Advanced Media Management
- Move media storage to IndexedDB/OPFS with streaming.
- Support inline video capture and screenshots.
- Global media library with tagging, search, and deduplication.

## 3. Intelligent Automation
- AI-assisted task summaries and priority suggestions.
- Smart note linking (automatic backlinks, graph view).
- Anomaly detection (overdue tasks, idle projects) with proactive nudges.

## 4. Integrations
- Two-way sync with Google Tasks/Calendar and Outlook.
- Zapier/IFTTT connectors for custom automations.
- Email-to-task gateway and mobile voice capture.

## 5. Reporting & Analytics
- Custom dashboards per workspace with shareable widgets.
- Time tracking insights (estimated vs actual, burn-down charts).
- Export to BI tools (CSV, JSON API, direct BigQuery/Sheets).

## 6. Offline & Sync Hardening
- Service-worker-based offline mode with conflict resolution UI.
- Sync health dashboard and activity log.
- Optional backend token broker for enterprise security.

## 7. Extensibility & Ecosystem
- Plugin system for custom task fields and automation rules.
- Scriptable command palette (user-defined macros).
- Public template gallery with community submissions.

## 8. Mobile & Desktop
- Dedicated mobile app using React Native or Capacitor.
- Desktop wrapper (Electron/Tauri) with offline cache.
- PWA install guidance and push notification support.

Have a new idea? Open an issue, drop it in the roadmap board, or append it here with a short rationale. We'll use this document as a seed list during planning sessions.
